# Contributing

Thanks for your interest in improving the Web3 Wallet Connect + Celo React Starter Kit.

## How to work on an issue

1. Fork this repository.
2. Create a feature branch from `main`.
3. Make small, focused changes.
4. Run the checks:
   - `npm run build`
5. Open a Pull Request with a clear description and, if relevant, screenshots.

## Ideas for contributions

- Improve wallet UX (loading states, error messages, network hints).
- Add more Celo examples (e.g. reading balances, simple contract calls).
- Enhance documentation (setup guides, troubleshooting, screenshots).

Please keep the code style consistent with the existing project and prefer TypeScript when adding new files.
